6.4: Parametric Tests for Two Populations

When we discussed parametric methods for modeling null distributions and calculating p-values, we focused on tests involving a single population. For example, we tested claims about a single mean and claims about a single proportion.

Is the mean pressure of North Atlantic storms less than 950 millibars?

Often we want to investigate questions about the association between two or more different variables. For example:

Is the mean pressure of North Atlantic storms less below the Tropic of Cancer than for North Atlantic storms above the Tropic of Cancer?

Are female faculty at universities and colleges in the United States paid differently than male faculty?

Does xylitol (a natural sweetener) helps prevent ear infections compared to no treatment?

Tests for Comparing Two Populations

When doing a hypothesis test for a difference between population means or two population proportions, we apply the same steps as with previous tests. The steps are listed below. Refer to Appendix A to review the steps.

  1. If when assessing claims comparing parameters, such as a difference, the hypothesis have the same general setup:
  • \(H_0\): There is no difference between the two groups.
  • \(H_a\): There is a difference between the two groups.

  1. Collect data from samples and calculate the difference in sample statistics.

  2. Calculate the p-value. Two possible methods are:

  • Using two-sample permutations tests.
  • Applying the Central Limit Theorem (CLT)

  1. Based on the significance level, \(\alpha\), make a conclusion.

  2. Clearly summarize the results.

A Test Statistic for Comparing Populations

After we have identified a test statistic, we can apply the CLT to standardize the test statistic and calculate p-values using the same strategy as the parametric approach for testing claims about a single population. We assume the claim in the null hypothesis is true, and then consider how likely is our sample test statistic. When comparing two populations, the null hypothesis is

  • \(H_0\): There is no difference. For example, \(\mu_1 - \mu_2 =0\) or \(p_1 -p_2 = 0\).

The standardized test statistic is the \(z\)-score or \(t\)-test statistic corresponding to the observed difference in sample statistics:

\[{\color{dodgerblue}{\mbox{standardized test stat} = \frac{ \mbox{difference in observed sample stats} - {\color{tomato}{\mbox{difference in null claim}}}}{\mbox{SE of Null Dist}} = \frac{ \mbox{diff in sample stats} - {\color{tomato}{0}}}{\mbox{SE of Null Dist}}}}.\]

To calculate the standardized test statistic, we need the following information:

  1. Do the samples satisfies conditions for CLT? If yes, continue on!
  2. What is the correct probability distribution to use, a \(t\)-distribution or a normal distribution?
  3. What is the difference in observed sample statistics?
  4. What is an estimate for the standard error of the null distribution?

Difference in Two Independent Means: A Parametric Method

If we are doing a test on a difference of means from two independent populations, then the null claim is \(H_0\): \(\mu_1 - \mu_2=0\) and recall the CLT for a difference in means is

\[\overline{X}_1 - \overline{X}_2 \sim N \left( {\color{tomato}{\mu_1 - \mu_2}} , {\color{dodgerblue}{\mbox{SE}}} \right) = N \left( {\color{tomato}{0}} , {\color{dodgerblue}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} } \right).\]

Population Variance is Known

  • We use a normal distribution to model the null distribution and calculate p-values.
  • We have the standardized test statistic

\[{\color{dodgerblue}{z = \frac{ \mbox{diff in sample stats} - {\color{tomato}{0}}}{\mbox{SE of Null Dist}} = \dfrac{(\bar{x}_1 - \bar{x}_2) - {\color{tomato}{0}} }{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}}}}.\]

Population Variance is Unknown

When testing claims about the difference in two population means, we typically do not know the population variances \(\sigma_1^2\) or \(\sigma_2^2\). As with a single mean, in the formula for the standard error, we plug-in sample standard deviations \(s_1\) and \(s_2\) in for \(\sigma_1\) and \(\sigma_2\), respectively.

  • We use a \(t\)-distribution instead of a normal distribution due to the added uncertainty with \(\sigma_1^2\) and \(\sigma_2^2\) unknown.

    • Many functions in R, such as t.test(), uses Welch’s approximation for the degrees of freedom (\(df\)).
    • More informally, we can approximate \(df\) using the smaller of either \(n_1-1\) or \(n_2 -1\).

  • We have the t-test statistic

\[{\color{dodgerblue}{{\color{tomato}{t}} = \frac{ \mbox{diff in sample stats} - 0}{\mbox{SE of Null Dist}} = \dfrac{(\bar{x}_1 - \bar{x}_2) - 0 }{\sqrt{\frac{{\color{tomato}{s_1}}^2}{n_1} + \frac{{\color{tomato}{s_2}}^2}{n_2}}}}}.\]

Using t.test() with Two Samples

Samples Stored as Separate Vectors

In R, we can use the command t.test(x1, x2, alt = [direction]).

  • Samples are stored in independent vectors x1 and x2.
  • Set the option alt based on the inequality used in \(H_a\).
    • Use "greater" for right-tail test.
    • Use "less" for left-tail test.
    • Use "two.sided" for a two-tail test.
    • If you do not indicate any alt option, the default is a two-tailed test.
  • The default is mu=0 which is exactly what we want, so we do not specify this option.

Subsetting Samples Inside t.test()

Frequently, we would like to compare the means of a quantitative variable, denoted quant, for two different levels of a categorical variable, denoted categorical, for sample data stored in a data frame called data.name. t.test() can subset the quant data into independent samples for us and perform a hypothesis test for the difference in two means:

t.test(quant ~ categorical, data = data.name, alt = [direction])

Difference in Two Proportions: A Parametric Method

If we are doing a test on a difference of two proportions, then the null claim is \(H_0\): \(p_1 - p_2=0\). We use the difference in sample means, \({\color{dodgerblue}{\hat{p}_1 - \hat{p}_2}}\), as the test statistic:

\[{\color{dodgerblue}{z = \frac{ \mbox{diff in sample stats} - {\color{tomato}{\mbox{diff in null claim}}}}{\mbox{SE of Null Dist}} = \dfrac{(\hat{p}_1 - \hat{p}_2) - {\color{tomato}{0}} }{\mbox{SE of Null Dist}}}}.\]

From the CLT for a difference in proportions we have

\[\widehat{P}_1 - \widehat{P}_2 \sim N \left( {\color{tomato}{p_1 - p_2}} , {\color{dodgerblue}{\mbox{SE}}} \right) = N \left( {\color{tomato}{0}} , {\color{dodgerblue}{\sqrt{\frac{p_1(1-p_1)}{n_1} + \frac{p_2(1-p_2)}{n_2}}} } \right).\]

Defining a Test Statistic

The null hypothesis is the claim \(p_1 - p_2 = 0\) or \(p_1 = p_2\). We have not claimed anything about the value of \(p_1\) on its own. We have not claimed anything about the value of \(p_2\). What can we substitute in place of \(p_1\) and \(p_2\) in the formula for the standard error? Let \(X_1\) and \(X_2\) denote the number of successes in samples 1 and 2 with sizes \(n_1\) and \(n_2\), respectively.

  • We use a normal distribution to model the null distribution (as long as both samples are large enough).
  • In \(H_0\) we assume there is no difference in the two groups, so we pool the samples together and calculate the sample pooled proportion:

\[{\color{tomato}{\mbox{sample pooled proportion} = \hat{p}_p = \frac{X_1 + X_2}{n_1 + n_2}}}.\] - To estimate the standard error, we use \({\color{tomato}{p_1 = p_2 = \hat{p}_p}}\). - The standardized test statistic is the z-score of the difference in sample proportions:

\[{\color{dodgerblue}{z = \dfrac{(\hat{p}_1 - \hat{p}_2) - {\color{tomato}{0}} }{\mbox{SE of Null Dist}}= \dfrac{(\hat{p}_1 - \hat{p}_2) - {\color{tomato}{0}} }{\sqrt{{\color{tomato}{\hat{p}_p}}(1 -{\color{tomato}{\hat{p}_p}})\left( \frac{1}{n_1} + \frac{1}{n_2} \right)}}}}.\]

Calculating the p-Value

When doing a hypothesis test on a difference of two proportions, we use a normal distribution to estimate the null distribution and calculate p-values. We can find the z-score of the observed difference in sample proportions, \(\hat{p}_1 - \hat{p}_2\), and then use a standard normal distribution to calculate the area in one or both of the tails. If we denote the z-score as z, then

  • For a left-tailed test, p-value = pnorm(z, 0, 1)
  • For a right-tailed test, p-value = 1 - pnorm(z, 0, 1)
  • For a two-tailed test, p-value = 2 * pnorm(-1*abs(z), 0 , 1)

Using the prop.test() Function

If one sample has size \(n_1\) with \(X_1\) successes and the other sample has size \(n_2\) with \(X_2\) successes, then we can use the prop.test() function in R as follows:

  • In R, enter the command prop.test(c(X1, X2), c(n1, n2), alt = [direction])
  • Use "greater" for the direction of a right-tail test.
  • Use "less" for the direction of a left-tail test.
  • Use "two.sided" for the direction of a two-tail test.
  • If you do not indicate any alt option, the default is a two-tailed test.

Appendix A: Steps for Performing a Hypothesis Test

  1. State the hypotheses and identify (from the alternative claim in \(H_a\)) if it is a one or two-tailed test.

    • \(H_0\) is the “boring” claim. Express using an equal sign \(=\).
    • \(H_a\) is the claim we want to show is likely true. Use inequality sign (\(>\), \(<\), or \(\ne\)).
    • State both \(H_0\) and \(H_a\) in terms of population parameters such as \(\mu_1-\mu_2\) and \(p_1-p_2\).

  2. Compute the test statistic.

    • If the observed sample contradicts the null claim, the result is significant.
    • A standardized test statistic measures how many SE’s the observed stat is from the null claim.
    • A standardized test statistic with a large absolute value is supporting evidence to reject \(H_0\).

  3. Using the null distribution, compute the p-value. The p-value is the probability of getting a sample with a test statistic as or more extreme than the observed sample assuming \(H_0\) is true.

    • The p-value is the area in one or both tails beyond the test statistic.
    • The p-value is a probability, so we have \(0 < \mbox{p-value} < 1\).
    • The smaller the p-value, the stronger the evidence to reject \(H_0\).

  4. Based on the significance level, \(\alpha\), make a decision to reject or not reject the null hypothesis

    • If p-value \(\leq \alpha\), we reject \(H_0\).
    • If p-value \(> \alpha\), we do not reject \(H_0\).

  5. Summarize the results in practical terms, in the context of the problem.

    • If we reject \(H_0\), this means there is enough evidence to support the claim in \(H_a\).
    • If we do not reject \(H_0\), this means there is not evidence to reject \(H_0\) nor support \(H_a\). The test is inconclusive.

Appendix B: Sample Sizes and the CLT

  • For a single mean, we can use the CTL to perform a hypothesis test as long as:

    • Either the population is symmetric or \(n \geq 30\).
    • If the sample is symmetric, we can assume the population is symmetric.

  • For a difference in two means , we can use the CTL to perform a hypothesis test as long as:

    • Population 1 is either symmetric or \(n_1 \geq 30\), and
    • Population 2 is either symmetric or \(n_2 \geq 30\).

  • For a single proportion, we can use the CTL to perform a hypothesis test as long as:

    • Both \(n\hat{p} \geq 10\) and \(n(1-\hat{p}) \geq 10\).

  • For a difference in two proportions, we can use the CTL to perform a hypothesis test as long as:

    • All of \(n_1\hat{p}_1 \geq 10\), \(n_1(1-\hat{p}_1) \geq 10\), \(n_2\hat{p}_2 \geq 10\), and \(n_2(1-\hat{p}_2) \geq 10\) are satisfied.
